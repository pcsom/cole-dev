`embedding_config.py`: Defines name, Huggingface Hub identifier, and type (transformer vs sentence-transformer) for each language model.

`generate_corpus.py`: Create a corpus file of all 5^6 NAS-Bench-201 architectures & performances.

`merge_onnx_corpus.py`: Merges a corpus of NAS-Bench-201 architectures generated by this codebase with a corpus of NAS-Bench-201 architectures generated by the ONNX-Net codebase. The join is performed by aligning test accuracies to a specified precision.

`stringify_utils.py`: Convert a NAS-Bench-201 architecture into a string

`embed_corpus.py`: Augment and convert a string into an LLM embedding

`robust_surrogate_predict.py`: The core comparison utility. Runs a surrogate performance comparison between two specified embedding configurations, by using a stratified subsampled cross validation.

`current_comp.py`: Runs a comparison experiment (calls the functionality present in `robust_surrogate_predict.py`).

`run.sh`: Sbatch submission script to run a comparison experiment.

`results_io.py`: Input/output (to file and print) management during experiments (pulling information from past trials, updating results, etc).

`stat_utils.py`: Statistical calculation and testing utilities.

`heads.py`: Defines regressor models (MLP, XGB, etc).

`softpca.py`: Custom PCA implementation that provides the option to perform 'whitening' on the data (unused at the time of paper submission)
